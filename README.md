# Tags

- **`<HTML>`** - Tag container raiz
  - DOM Interface
  - Uses HTMLElement  

- **`<HEAD>`** - Cabeça da página
  - DOM Interface
  - Uses HTMLElement
  - Contido por: `<HTML>` 

- **`<TITLE>`** - Título da página
  - DOM Interface
  - Uses HTMLElement
  - Categoria = Metadado
  - Contido por: `<HEAD>`

- **`<BODY>`** - Corpo da página
  - DOM Interface
  - Interface HTMLBodyElement:HTMLElement
  - Categoria = Seção Raiz
  - Contido por: `<HTML>`
